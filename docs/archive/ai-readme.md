# AI Assistant Guide - ИТТСУ Лендинг

## О проекте
**Институт:** ИТТСУ (Институт транспортной техники и систем управления) РУТ  
**Прежнее рабочее название:** ИЖТ РУТ  
**Цель лендинга:** Презентация специальностей и навигация абитуриентов

## Задача
- Красиво отобразить все специальности института (01-33)
- Предоставить информацию о каждой специальности
- Перенаправить на:
  - Госуслуги (подача документов)
  - Приемную комиссию (очно по карте)
  - Контакты приемной комиссии (телефон/почта)

## Дизайн-система
Смотри `style-guide.md` - основные цвета, шрифты и стилистические решения

## Технологии
- HTML5
- CSS3 (vanilla)
- JavaScript (vanilla)
- SVG для графики

## Функциональные требования
1. **Адаптивность:** мобильная, планшетная, десктоп версии
2. **Интерактивность:** 
   - Открытие детальной информации по специальности
   - Плавные анимации переходов
   - Навигация между специальностями
3. **Производительность:** оптимизация изображений, lazy loading

## Целевые действия пользователя
1. Подать документы через Госуслуги
2. Найти приемную комиссию на карте
3. Связаться по телефону/почте

## Формат файлов
В исходной папке `update/сurriculums`, каждая специальность:
- **xx-cover.png/jpeg:** Шапка карточки специальности
- **xx-professions-bg.png/jpeg:** Фон блока профессий
- **xx.md:** Текстовая информация в Markdown
- **exampleXX.svg:** Примеры реализованных карточек
